Quantified Self and Wearable Data Analysis

Submitted by: Ayush Kesharwani

Date: April 2026

1. Objective

To analyze personal health data collected from wearable devices using clustering algorithms and predictive ML models to generate health insights and predict disease risk.

2. Dataset Overview

FeatureDescription
Daily StepsTotal steps walked per day
Heart RateAverage BPM
Sleep HoursTotal sleep per night
Active MinutesMinutes of physical activity
Calories BurnedDaily caloric expenditure
BMIBody Mass Index
Stress LevelScale 1-10

3. Methodology

3.1 Clustering (K-Means)

Applied K-Means clustering with K=4 (selected via Elbow Method). PCA used for 2D visualization.

3.2 Disease Risk Prediction

Random Forest and Gradient Boosting classifiers trained to predict high/low disease risk.

4. Results

4.1 Cluster Results

Sedentary: 126 users (25.2%)

4.2 Model Performance

6. Conclusion

The K-Means clustering successfully identified 4 distinct user health profiles. The Random Forest model achieved high accuracy in predicting disease risk, with BMI, sedentary minutes, and daily steps being the most important features. These insights can help in personalizing health recommendations for wearable device users.
